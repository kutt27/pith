What is Pith?

Pith is an ultra-lightweight UI framework that styles native HTML elements directly.

Why use Pith?

Pith is small, fast, and easy to use. No complex build tools or dependencies required.

How do I get started?

Just include the CSS and JS files in your HTML. Then write normal semantic HTML.
